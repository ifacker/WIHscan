## 简介：
WIHscan 主要扫描网站 js 是否存在敏感信息，包括一些 AK/SK 之类的，大家可以去 config/ 中自定义规则，从而匹配出自己想要的结果（需要会点正则表达式）  

🎉🎉🎉！
## 使用方法：
```shell
./WIHscan -u http://www.xxx.com/xxx.js
./WIHscan -f jsfile.txt
./WIHscan -fj ARLxxx.json
```
